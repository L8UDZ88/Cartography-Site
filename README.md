# Cartography Site

The marketing site for **Cartography** — the skill of building timeless navigational maps on every level — and its brand network.

> Remember who you are. Map where you stand. Chart the way forward.

## The Network

| Brand | Axis | Promise |
|---|---|---|
| **Legendary Lessons** | Free / Feeder | The timeless playbook, one legend at a time |
| **GrowFast** | Revenue · Wealth | Map. Measure. Manage. Predictable revenue |
| **OmniStory** | Reality · Health | Map the story before it maps you |
| **Attraction Factors** | Relationships · Happiness | Become the one the right things move toward |

## Pages

- `index.html` — Cartography homepage (hub linking to all four brands)
- `legendary-lessons.html`
- `growfast.html`
- `omnistory.html`
- `attraction-factors.html`
- `styles.css` — shared stylesheet

## Tech

Plain HTML + CSS. No build step, no dependencies. Just open `index.html` in any browser, or host the folder anywhere (GitHub Pages, Netlify, Vercel, etc.).

## Run locally

Double-click `index.html`, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Publish free with GitHub Pages

1. Push this folder to a GitHub repo named **Cartography Site**.
2. In the repo: **Settings → Pages → Source: `main` branch / root**.
3. Your site goes live at `https://<your-username>.github.io/Cartography-Site/`.

## Notes

- The email capture forms are wired for demo only. Connect them to your email tool (Beehiiv, etc.) to go live.
